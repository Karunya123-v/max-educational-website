# MAX Educational Council Website

This project is a responsive and interactive website developed for MAX Educational Council, an educational consultancy that guides students in choosing the right academic path based on their interests.

## Features
- Clean and modern user interface
- Course guidance for Engineering, Management, Arts, and Science
- Student registration form
- Responsive design for mobile and desktop
- Smooth navigation and basic interactivity

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
The main objective of this project is to provide an easy-to-use platform where students can explore different educational fields and get assistance in admission processes.

## Future Enhancements
- Backend integration for real-time data
- Login and admin dashboard
- Database connectivity
